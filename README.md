## üìÇ Project Overview

The dataset contains detailed transactional information related to global bike sales.  
Key fields include:

- **Date, Month, Year**
- **Customer Age & Age Group**
- **Customer Gender**
- **Country & State**
- **Product Category & Sub-Category**
- **Order Quantity**
- **Unit Cost, Unit Price, Profit, Revenue**

Using this data, I built clean, well-formatted Excel charts to communicate insights effectively.
